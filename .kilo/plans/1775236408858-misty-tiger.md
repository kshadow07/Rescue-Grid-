## Phase 5 Tasks

### 5.1 ‚Äî Create Assignment Modal
- Build `components/dma/CreateAssignmentModal.tsx`
- Triggered by `+ CREATE TASK` topbar button or `ASSIGN ‚Üí` from report pin popup
- Pre-fill `linked_report_id` when opened from pin popup
- Fields:
  - `TASK DESCRIPTION` textarea
  - `LOCATION` using Mapbox SearchBox (`@mapbox/search-js-react`): live suggestions ‚Üí on select auto-fills `location_label` + hidden `latitude`/`longitude`
  - `URGENCY` select: Critical / Urgent / Moderate
  - `ASSIGN TO` radio: "Individual Volunteer" OR "Task Force" (mutually exclusive)
    - Volunteer dropdown: `volunteer WHERE status='active'`
    - Task Force dropdown: `task_force WHERE status='active'`
  - `TIMER (optional)` datetime-local input
  - `LINKED REPORT (optional)` dropdown: open victim reports
- `CREATE ASSIGNMENT` primary button ‚Üí POST `/api/dma/assignment`
- `CANCEL` ghost button ‚Üí closes modal

### 5.2 ‚Äî API Route: POST /api/dma/assignment
- Validate: `task`, `urgency`, `location_label`, `latitude`, `longitude` required ‚Üí 400 if missing
- Enforce: exactly one of `assigned_to_volunteer` OR `assigned_to_taskforce` ‚Üí 400 if both or neither
- Insert into `assignment` with `status = 'active'`
- If `assigned_to_volunteer`: fetch `push_token`, send Web Push via `lib/push/sendPush.ts`
- If `assigned_to_taskforce`: fetch all member `push_token` via join, send push to each
- Return created row with 201

### 5.3 ‚Äî Assignments List (`/dma/assignments`)
- Build `app/(dma)/dma/assignments/page.tsx`
- Card list sorted `created_at DESC`
- Each card: task (truncated), urgency badge, status badge, assignee name, location_label, timer countdown if set
- Click card ‚Üí expand: full task, linked report, action buttons
- `MARK COMPLETED` ops Button + `MARK FAILED` danger Button ‚Üí PATCH `/api/dma/assignment/[id]`

### 5.4 ‚Äî API Route: PATCH /api/dma/assignment/[id]
- Accept `{ status: 'completed' | 'failed' }`
- Update `assignment.status` and `assignment.updated_at = now()`
- If `completed` and `victim_report_id` is set: also update `victim_report.status = 'resolved'`
- Return updated row
